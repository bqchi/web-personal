Sistema Integrado de Relevamiento Prestacional

Solución Integral para el relevamiento prestacional médico de una importante obra social.

Integra el uso de macros y formularios en Excel, para la recolección de los datos. A esos datos se los pasa por un script de Python que automatiza la generación de coordenadas geográficas para cada registro. Por ultimo, la información generada es publicada en una Tabla de Prestadores de uso interno, el cual contiene una tabla general con varios filtros y un mapa donde se pueden observar los prestadores distribuidos por las Provincias.
